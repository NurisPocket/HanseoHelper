# 📝 HanseoHelper

학교 학생들을 위한 다목적 심부름 앱 서비스

<br/>

## 🧑🏻‍💻 Feature

- 성별, 선호 색상, 옷의 종류를 입력받아 코디 자동 추천, 추천받은 옷을 바로 구입할 수 있는 사이트로 이동할 수 있는 링크 제공, 옷을 추천받고 별점과 후기를 작성 가능

<br/>

## :pencil2: What i take on

- 버튼에 ID값을 부여하여 파일 속에 있는 랜덤 이미지 파일 출력 구현
- 알고리즘을 통한 코디 자동 추천 서비스 구축
